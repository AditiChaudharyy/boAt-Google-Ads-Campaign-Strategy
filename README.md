# boAt Google Ads Campaign Strategy 2026

## Project Overview
Full-funnel Google Ads simulation for boAt Lifestyle — 
India's #1 audio brand.

## Tools Used
- Google Trends
- SEMrush
- Ubersuggest
- Google Data Studio
- Google Slides

## Deliverables
- 50 keywords across 3 campaign stages
- 25 negative keywords
- 3 RSA ads with A/B test framework
- Live dashboard
- Optimisation report
